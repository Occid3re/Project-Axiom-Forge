2. Nature of the Service

LostUplink: Axiom Forge is a purely read-only simulation viewer. There is no login, no account creation, no user-generated content, and no interactive input of any kind. Visitors passively observe a continuously running server-side evolutionary simulation.

3. Data We Process

  • Standard HTTP access logs (IP address, timestamp, requested URL, browser user-agent) retained briefly for security and error analysis
  • WebSocket connection metadata (connection and disconnect timestamps) held in memory only for the duration of the session
  • No personal data is actively collected, stored, or shared with third parties

4. Legal Basis

  • Legitimate interest for stable operation, security monitoring, and error analysis (Art. 6(1)(f) GDPR)

5. Storage

  • Access logs are retained for up to 14 days for security purposes and then deleted automatically
  • WebSocket session data is held in memory only and discarded when the connection closes
